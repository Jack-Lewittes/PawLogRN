# What's in the app? 
<br>
A mobile app to track your puppies activities from potty to training to nap time
<br>
  - Onboarding flow (dog name + your profile)<br>
  - Home tab: today's log, running nap/training banners, FAB with all 7 activity types<br>
  - Log tab: date navigator + full timeline with meal badges and feeding details<br>
  - Profile tab: dog info + weight tracking<br>
  - Settings tab: multi-user support, switch who's logging<br>
