How to run PawLog on your iPhone                                                                                                                                                                                                
                                                                                                                                                                                                                                            
  Step 1 — Install Expo Go on your iPhone                                                                                                                                                                                                 
  Search "Expo Go" in the App Store and install it (free).                                                                                                                                                                                  
                                                                                                                                                                                                                                            
  Step 2 — Open the project in Cursor/VS Code                                                                                                                                                                                               
  File → Open Folder → Desktop/MacBook/PawLogRN                                                                                                                                                                                             
                                                                                                                                                                                                                                            
  Step 3 — Open a terminal and start the dev server                                                                                                                                                                                         
  cd /Users/jack.lewittes/Desktop/MacBook/PawLogRN
  npx expo start

  Step 4 — Open on your iPhone
  A QR code appears in the terminal. Open your iPhone's Camera app, point it at the QR code, tap the banner that appears — Expo Go opens and loads PawLog.

  That's it. Every time you save a file in Cursor, the app updates live on your phone.

  ---
  What's in the app

  - Onboarding flow (dog name + your profile)
  - Home tab: today's log, running nap/training banners, FAB with all 7 activity types
  - Log tab: date navigator + full timeline with meal badges and feeding details
  - Profile tab: dog info + weight tracking
  - Settings tab: multi-user support, switch who's logging
